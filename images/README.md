# Images Directory

Place your profile photo here.

## Profile Photo Requirements

- **Filename**: `profile.jpg` (or update the path in `data/content.json`)
- **Recommended Size**: 400x400 pixels or larger
- **Format**: JPG or PNG
- **File Size**: Under 500KB for fast loading
- **Aspect Ratio**: Square (1:1) works best

## Tips for a Great Profile Photo

1. Use a professional, high-quality photo
2. Good lighting and clear face visibility
3. Neutral or academic background
4. Professional attire appropriate for your field
5. Smile naturally

## Example

After adding your photo, the structure should look like:
```
images/
├── profile.jpg       # Your photo
└── README.md         # This file
```

You can also add other images here if needed for projects or publications.
